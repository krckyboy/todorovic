# Vision: todorovic Portfolio

## Purpose

Create a personal portfolio and blog that showcases professional work, shares knowledge through blog posts, and provides a point of contact for potential collaborators.

## Target Audience

1. **Potential Employers/Clients**
   - Looking for evidence of technical skills
   - Want to see real projects and thinking
   - Need easy way to contact

2. **Fellow Developers**
   - Interested in technical approaches
   - Looking for tutorials/guides
   - May want to collaborate

3. **Blog Readers**
   - Interested in web development topics
   - Looking for practical solutions
   - Want clear, actionable content

## Goals

### Primary

1. Showcase technical competence through projects and writing
2. Share knowledge via blog posts
3. Provide professional contact point

### Secondary

1. Experiment with modern web technologies
2. Demonstrate AI-assisted development workflow
3. Build in public (document the process)

## Success Metrics

| Metric                   | Target |
| ------------------------ | ------ |
| Lighthouse Performance   | 95+    |
| Lighthouse Accessibility | 100    |
| Time to Interactive      | < 2s   |
| Blog posts per month     | 2+     |
| Build time               | < 30s  |

## Design Principles

1. **Simplicity** - Clean, focused design
2. **Speed** - Fast loading, minimal JS
3. **Accessibility** - Usable by everyone
4. **Content-first** - Text is primary
5. **Progressive** - Works without JS

## Technical Philosophy

- **Static over dynamic** - Generate at build time
- **HTML over JS** - Use platform features
- **CSS over JS** - Styling without runtime
- **Markdown over CMS** - Simplicity over features
- **AI-assisted** - Use tools to enhance productivity

## Non-Goals

- Social features (comments, likes)
- User authentication
- Real-time features
- E-commerce
- Heavy animations
