## ADDED Requirements

### Requirement: Blog authoring guidance includes tag governance

The blog authoring workflow documentation SHALL include explicit tag governance guidance aligned with the canonical taxonomy.

#### Scenario: Governance guidance is documented

- **WHEN** reading `documentation/blog-authoring-workflow.md`
- **THEN** canonical tags are listed with usage intent
- **AND** authoring constraints are documented (count, naming, new-tag policy)
- **AND** event-style tag guidance is explicitly defined
- **AND** examples of good vs poor tag choices are included
